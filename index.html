Política de Privacidade

Introdução

O nosso aplicativo é comprometido em proteger a privacidade e a segurança dos dados dos nossos usuários. Esta política de privacidade descreve como coletamos, usamos e protegemos os dados dos nossos usuários.

Coleta de Dados

Nós coletamos os seguintes dados dos nossos usuários:

  • Nome e sobrenome
  • E-mail
  • Informações de login (username e senha)

Uso de Dados

Nós usamos os dados coletados para:

  • Fornecer os serviços do aplicativo
  • Melhorar a experiência do usuário
  • Proteger a segurança e a integridade do aplicativo

Proteção de Dados

Nós protegemos os dados dos nossos usuários com medidas de segurança adequadas, incluindo:

  • Criptografia de dados
  • Autenticação de usuários
  • Controle de acesso

Compartilhamento de Dados

Nós não compartilhamos os dados dos nossos usuários com terceiros, exceto quando necessário para fornecer os serviços do aplicativo.

Direitos dos Usuários

Os usuários têm o direito de:

  • Acessar e corrigir seus dados
  • Cancelar sua conta
  • Exercer seus direitos de acordo com a legislação aplicável

Mudanças nesta Política

Nós nos reservamos o direito de fazer alterações nesta política de privacidade a qualquer momento, mediante comunicação aos nossos usuários.
